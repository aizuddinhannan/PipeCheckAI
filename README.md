# PipeCheckAI

AI-Powered Underground Utility Quality Assurance System

## Overview

PipeCheckAI is a desktop application developed to automate the Quality Assurance (QA) process for Underground Utility Detection & Mapping (UDM) drawings.

The system is designed to assist surveyors, engineers, utility consultants, and QA teams by automatically checking CAD drawings against engineering standards and generating professional QA reports.

---

## Objectives

- Reduce manual QA time
- Improve drawing consistency
- Detect drafting errors automatically
- Standardize QA procedures
- Generate professional QA reports

---
